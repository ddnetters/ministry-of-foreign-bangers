# Ministry of Foreign Bangers

Dutch government style [website](https://ddnetters.github.io/ministry-of-foreign-bangers/) for a spotify playlist.

## Project setup

```bash
npm install
npm run build
```

## Project structure

```bash
/dist # Compiled files
/src  # Source files
```

## Deployment

```bash
git subtree push --prefix dist origin gh-pages
```
